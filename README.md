ForoHub API es una API REST desarrollada con Java 17 y Spring Boot, diseñada para gestionar un sistema de foros de discusión. Permite a los usuarios crear tópicos, responder discusiones, gestionar cursos y administrar usuarios mediante un sistema de autenticación segura con JWT y control de roles (USER, MODERATOR, ADMIN).

La aplicación implementa buenas prácticas de desarrollo backend como arquitectura en capas, DTOs, validaciones, manejo global de excepciones, paginación, borrado lógico y migraciones de base de datos con Flyway.

Incluye funcionalidades como:

Gestión completa de tópicos y respuestas

Sistema de soluciones en discusiones

Administración de cursos por categorías

Autenticación y autorización con JWT

Documentación automática con Swagger

Proyecto desarrollado por Mario Sinigalle como práctica de desarrollo backend con Spring Boot y arquitectura RESTful.
